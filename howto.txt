branches: master - all files
          github - subset that will be pushed to git hub (lo llam√© publish en pc3)

Add files to subset:

git checkout github
git checkout master file-name
git commit

Drop files from subset:

git checkout github
git rm -rf file-to-drop
git commit

To push subset to 'public_repo':

git push public_repo github:master
